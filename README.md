## 1️⃣ What is the difference between var, let, and const?

## Difference between var, let, and const:

## var

.Function scoped

.Can be redeclared

.Value can be reassigned

## let

.Block scoped

.Cannot be redeclared

.Value can be reassigned

.Introduced in ES6

## const

.Block scoped

.Cannot be redeclared

.Value cannot be reassigned

.Must be initialized when declared

## 2️⃣ What is the spread operator (...)?

Spread operator (...) is used to expand arrays or objects into individual
elements.

## 3️⃣ What is the difference between map(), filter(), and forEach()?

map(): Works on each element of an array and returns a new array.

filter(): Works based on a condition and returns a new array with elements that
satisfy the condition.

forEach(): Works on each element of an array. It does not return a new array. It
is used only to run a loop.

## 4️⃣ What is an arrow function?

An arrow function is a shorter way to write a function in JavaScript using the
=> syntax.

## 5️⃣ What are template literals?

Template literals are a way to write strings in JavaScript using backticks ( )
and insert variables with ${}.
